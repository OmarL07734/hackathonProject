Before we Begin

This tutorial uses github as its web host. The url will be controlled by Github and will look something like https://githubUserName.github.io/nameOfReprository/. If you do have a custom domain you can set the url to go to your domain through github. This is just a beginner guide into hosting a website, it will not go into detail about coding your website, it will just be about hosting the code that you already written.

Setting Up a Repo
  • First in any page in github, in the top right corner click the "+" and click on new repo
  • Add a name for the repo (this will be apart of the default url) and then add a description to describe the website
  • Next you would want to set the repo to public as the github pages will not work with any other settings
  • Optionally you can add a readme file which allows you describe your website to other github users looking at your code
  • Now click create
Adding Code To Your Github Repo

After adding your code to your repo, you must do a couple more things before you can see it live.

  • You have to go to your repo settings
  • In the left side of your screen you have to click pages
  • Last thing is to set your branch to main

After completing these steps a url should appear above your branch selection. Any changes you do to your code in the repo will be reflected on that website url after sometime.
